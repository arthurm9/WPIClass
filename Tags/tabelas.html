Como fazer tabelas

  1. Toda tabela tem que começar com a tag table
  2. A Tabela é dividida em três partes: thead, tbody, tfoot
  3. Somente tbody é obrigatória
  4. Para cada linha da Tabela usamos a tag tr
  5. As colunas possuem dois tipos: th para títulos e td para o corpo
